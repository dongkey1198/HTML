Anchor: 링크

설명:

현위치에서 다른 위치로 이동할때 사용된다. Anchor 태그는 다른 태그들과 다르게 a 태그안에 href 라는 Attribute를 추가해야한다. href란 "HyperText Reference"의 줄인말으로써 html문서의 주소를 의미한다. 즉, 현페이지에서 다른 페이지로 이동하거나 같은 페이지 내에서 다른 부분으로 이동할때 사용된다. 이때는 이동하고자하는 위치의 id값을 이용하여 이동할수있다.

활용 분야:

    1. 웹 URL
    2. 페이지 내 이동
    3. 메일주소
    4. 전화번호
