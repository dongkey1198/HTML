Lists: 목록

설명:

목록을 표시할때 사용되는 테그들이다. 종류로는 ol, ul이 존재한다. ol = ordered list 는 순서가 중요한 목록에 사용된다. ul = unordered list는 순서가 중요하지 않은 목록에 사용된다. 이 두 테그 모두 리스트의 항목을 표시할때는 list item이라는 테그를 사용한다. 이때 리스트 테그들의 자식 테그로는 li이만 사용할수있다.

개발분야:(ul 예제)

  • 웹 개발자
  • 서버 개발자
  • 프론트엔드 개발자
  • 백엔드 개발자
  • 자바 개발자
  • 안드로이드 개발자
  • ios 개발자
