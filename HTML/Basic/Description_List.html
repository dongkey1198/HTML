Description List: 정의 목록

설명:
1. 용어를 정의할 대 사용하는 저의 리스트
2. key-value로 정보를 제공할때 사용된다.

Description List 관련 테그들:
dl = description list
dt = description term(key값의로 사용된다)
dd = description data(value값의로 사용된다.)
즉 dl태그는 dt태그와 dd태그를 함께 사용해야한다.
